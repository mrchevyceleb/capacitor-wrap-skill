## Recommended Pricing Guidelines

### Subscriptions
- **Monthly:** $4.99 - $9.99 (most common for consumer apps)
- **Yearly:** $39.99 - $79.99 (20-30% savings vs monthly)
- **Free Trial:** 3-7 days (recommended to drive conversions)

### Consumables
- **Starter Pack:** $4.99 (50-100 credits)
- **Value Pack:** $14.99 (200-300 credits)
- **Best Value:** $29.99 (500-1000 credits)

**Pro Tip:** Offer 3 tiers to leverage the "goldilocks effect" - most users choose the middle option.

## Key Features

### Product ID Consistency
All documentation emphasizes that product IDs must match EXACTLY across:
1. Code (TypeScript constants)
2. RevenueCat dashboard
3. App Store Connect (iOS)
4. Google Play Console (Android)

### Platform-Specific API Keys
Clear guidance on using separate RevenueCat API keys:
- iOS: `appl_...` prefix
- Android: `goog_...` prefix

Both stored in environment variables and selected based on platform at runtime.

### Complete Testing Procedures

**iOS Sandbox Testing:**
- Create sandbox tester accounts
- Sign out of real App Store account
- Test purchases without real charges
- Verify RevenueCat dashboard events

**Android License Testing:**
- Add tester emails to license testing
- Instant, free test purchases
- Verify events in RevenueCat

### Common Pitfalls Documentation

Based on real-world experience, documented 10 common issues:
1. Product ID mismatches
2. Products not approved (iOS)
3. Products not activated (Android)
4. Wrong API keys
5. Sandbox account not signed in
6. Subscription groups not set
7. Receipt validation failures
8. Free trial issues
9. Currency conversion problems
10. Webhook not firing
