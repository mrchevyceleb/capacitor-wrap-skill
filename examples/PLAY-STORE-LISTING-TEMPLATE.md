## Monetization Products

### In-App Purchases Checklist

If your app includes subscriptions or consumables (credits, tokens, etc.), complete these steps:

**Subscription Setup:**
- [ ] Navigate to: **Monetize** → **Products** → **Subscriptions**
- [ ] Create subscriptions with exact product IDs from code
  - Example: `pro_monthly` - $4.99/month
  - Example: `pro_yearly` - $39.99/year
- [ ] Set up base plans with billing periods
- [ ] Configure prices (USD auto-converts to other currencies)
- [ ] Optional: Enable free trial (7 days recommended)
- [ ] **Activate** each subscription (top right button)

**Consumable Products Setup:**
- [ ] Navigate to: **Monetize** → **Products** → **In-app products**
- [ ] Create consumables with exact product IDs from code
  - Example: `credits_50` - $4.99
  - Example: `credits_200` - $14.99
  - Example: `credits_500` - $29.99
- [ ] Set names and descriptions
- [ ] Configure prices for each product
- [ ] Set status to **Active**

**Product ID Naming Conventions:**
- Subscriptions: `{tier}_{period}` (e.g., `pro_monthly`, `premium_yearly`)
- Consumables: `credits_{amount}` or `tokens_{amount}` (e.g., `credits_50`)
- Must match EXACTLY with your code and RevenueCat dashboard

**Recommended Pricing:**
- Monthly subscription: $4.99 - $9.99
- Yearly subscription: $39.99 - $79.99 (20-30% savings)
- Starter credits: $4.99
- Value pack: $14.99
- Best value: $29.99
